# Cartolink Internship Assessment

A single page UI designed to show my skill for internship role at Cartolink

## Tech Stack

- Next.js
- TypeScript
-React Icons

## Project Structure

- `app` - Next.js app directory
- `hooks` - Custom hooks for reusable logics
- `components` - Reusable UI components
- `lib` - Utility functions and constants
- `public` - Static assets
- `contents` - Global array for data fetching
